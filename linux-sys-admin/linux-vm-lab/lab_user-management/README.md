# Lab: User and Group Management

## Objective
Practicing creating and managing users and groups in Linux and understand how permissions affect access.

## Environment
- OS: Ubuntu
- VM: VirtualBox
- User privilege level: sudo

## Scenario
A new employee needs an account to access active directories, while another employee will be losing permissions

## Tasks Performed
- Created new users and groups
- Assigned users to groups
- Modified directory permissions
- Verified access control

## Commands
See commands.md

## Outcome
- User access was correctly implemented for new employee
- User access was correctly restricted
- Group-based permissions worked as expected

## What I Learned
- Importance of priveleges
- How group permissions simplify access control
